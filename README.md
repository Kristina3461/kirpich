# Веб-приложение «Магазин крафтового пива Кирпич»
## Описание
Этот проект представляет собой веб-приложение для магазина крафтового пива "Кирпич".
### Технологии
- Python 3.10
- Django
### Запуск проекта
- python manage.py runserver
